Detailed Description

Text preprocessing utilities for transformer input.

The TextPreprocessor class provides static methods for preparing text data before tokenization and model input. Features include:

  • Case normalization
  • Training data pair processing
  • String cleaning and normalization
  • Batch processing capabilities
